# Health Website - Static Hosting on AWS S3

# Project Overview
This is a static website focused on **health awareness** (nutrition, fitness, wellness, etc.). The website was built using pure HTML, CSS, and JavaScript. It was successfully hosted as a static website on Amazon S3 as part of the AWS assignment.

## Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript**
- **Amazon S3** (Static Website Hosting)

# Live Demo
🔗 [View Live Website](http://my-first-s3-bucket-project19.s3-website-us-east-1.amazonaws.com/)


# Project Features
- Responsive design
- Clean and modern UI for health-related content
- Interactive elements powered by JavaScript
- Fully static website (no backend required)

# What I Accomplished
1. Downloaded the health website template from tool.com
2. Created a new S3 bucket in AWS
3. Uploaded all website files (HTML, CSS, JS, images, etc.) to the S3 bucket
4. Enabled Static website hosting in the bucket properties
5. Set `index.html` as the Index document
6. Configured the bucket to allow public read access using a Bucket Policy
7. Successfully hosted the website and made it publicly accessible via the S3 website endpoint

# How to Run Locally
1. Clone or download this repository
2. Open the `index.html` file in any modern web browser
3. No installation or server required
